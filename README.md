# Web Bán Giày

Website bán giày được xây dựng bằng React, TypeScript và Vite.

## Công nghệ

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Radix UI / shadcn-style components

## Cấu trúc chính

```text
src/
├─ main.tsx          # Điểm khởi chạy ứng dụng
├─ app/              # Routes, pages, components, context, data
├─ imports/          # Giao diện và asset export từ Figma
└─ styles/           # CSS, theme, font, Tailwind
```

## Cài đặt

```bash
npm install
```

## Chạy development

```bash
npm run dev
```

## Build

```bash
npm run build
```
